# Pomodoro Timer App

Keep track of your coding sessions.

## Getting Started

Clone repo down to your machine.

Change into API directory

`cd pomodoro-app/pomodoro-api`

Run `bundle install`

Run migrations

`rake db:migrate`

Run server

`rails server`

Change into client directory

`cd ../pomodoro-client`

Open index.html in browser

`open index.html`
